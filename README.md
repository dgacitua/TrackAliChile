# TrackAliChile

Visitar página: http://trackalichile.herokuapp.com

## Descripción

Aplicación web para rastrear paquetes de Aliexpress Standard Shipping y Correos Chile.

Creada por Daniel Gacitúa. Liberada como software libre mediante AGPLv3.

Esta aplicación no es oficial. Toda la información obtenida por la aplicación está disponible a través de APIs públicas y de libre acceso. Las marcas e instituciones mencionadas en esta aplicación pertenecen a sus respectivos dueños.
